建立本地库并通过git上传至github

2018-11-01

1、本地项目目录创建本地仓库

git init

输入命令后项目目录会有一个隐藏的.git文件夹

2、上传所有代码至本地仓库

git add .

3、提交代码至本地仓库

git commit -m "initial commit"

4、上传代码到github

4.1 新建repository

4.2 关联远程仓库

git remote add origin https://github.com/sunnywp/Biology.git

4.3 关联远程仓库

git push origin master
