基于Hexo搭建个人博客

一直以来，都想拥有属于自己的一个站点，用来记录生活的点滴以及学习的历程。
也是偶然的机会，看到同事的博客，体验甚好，于是决定将想法付诸行动。
过去的两天，参考同事的博客，基于Hexo搭建个人博客，少走了很多弯路。自然，过程中也遇到一些问题，故总结一下。

一、前置条件

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown解析文章，在几秒内，即可利用靓丽的主题生成静态网页。安装 Hexo 相当简单。但是在安装前需要检查电脑中是否已安装下列程序：

  • Node.js
  • Git

具体的安装可以参考百度，在此不再赘述。

二、Github上建立blog仓库

github有一个很有爱的项目，叫做github pages，这个项目是给开发者建立一个私人页面，上面用来分享新颖的想法和自己写的代码。个人博客可以基于github pages，省去很多麻烦。

创建博客仓库

创建和用户名对应的仓库，比如github用户名时sunnywp,仓库名就叫sunnywp.github.io。这一点需要切记，否则无法通过sunnywp.github.io进行访问。
